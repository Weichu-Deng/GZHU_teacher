# 中山大学爬取
使用`Selenium`+火狐浏览器可以绕过反爬机制
## selenium安装
```
pip install selenium==4.1.3
```
## 火狐浏览器安装
1. 安装火狐浏览器
2. firefox路径添加环境变量
3. 下载geckodriver驱动
`https://github.com/mozilla/geckodriver/releases`
下载完成后把geckodriver.exe放在firefox的安装路径下
4. 在火狐浏览器安装Selenium IDE 插件

**配置完运行`sysu_cs.py`脚本**
